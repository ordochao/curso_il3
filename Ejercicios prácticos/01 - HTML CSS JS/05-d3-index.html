¡Primer ejemplo del curso con D3!

Cacharrea desde la consola todo lo que quieras

Este ejemplo tiene cargada la librería d3. Ve a la consola y prueba los comandos que quieras. Aprende a seleccionar elementos HTML y SVG, por ejemplo -> d3.select("p") seleccionará el primer parrafo de tu página.

Cambia sus atributos y estilos. Puedes modificar el color del texto o del fondo. También los tamaños de fuente y la posición. ¡Dale caña!
